# Projeto Cuphead DLC — Página Temática

Este projeto apresenta uma página temática inspirada na última DLC do jogo Cuphead. O site contém informações, imagens e um design coerente com o estilo artístico do jogo.

## Demo
Acesse o projeto através do link:  
https://garygarcia2703.github.io/Cuphead-dlc/

## Descrição do Projeto
A proposta é recriar uma página informativa com foco visual, utilizando elementos gráficos e uma estética que remete ao estilo clássico inspirado em desenhos animados dos anos 1930. O site fornece uma visão geral da DLC por meio de seções organizadas.

## Funcionalidades
- Página informativa sobre a DLC
- Estrutura visual inspirada na estética do jogo
- Seções organizadas apresentando conteúdo e imagens

## Tecnologias Utilizadas
- HTML5  
- CSS3  

## Objetivo do Projeto
Praticar:
- Criação de páginas temáticas
- Uso de imagens e elementos visuais
- Organização de conteúdo estático com foco estético
